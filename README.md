# 🏡 Property Investment Calculator

## 📌 Overview
This is a **Flask-based Property Investment Calculator** developed for **CPS3962**.  
It allows users to **calculate different financial aspects** of property investments, including:
- **Acquisition Costs** (Purchase Price, Closing Costs, etc.)
- **Operating Expenses** (Homeowners Insurance, Property Tax, etc.)
- **Cash Flow Analysis** (Rent Revenue, CoC Return Goal)
- **Annual Growth Projections** (Rent Growth, Appreciation, etc.)


- ## 🔧 Installation & Setup
- ### **1️⃣ Clone the Repository**
```sh
git clone https://github.com/YOUR_USERNAME/PropertyCalculator.git
cd PropertyCalculator

```
## 2️⃣ Install Dependencies**

Make sure you have **Python 3** and `pip` installed, then install Flask:

```zsh
pip install flask
```


### **3️⃣ Run the Application**
`python app.py`
